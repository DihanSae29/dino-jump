# 🦕 Dino Jump

A tiny pixel-art endless runner built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

**[▶ Play Now](https://DihanSae29.github.io/dino-jump/dino_goreng.html)**

---

## Controls

| Input | Action |
|-------|--------|
| `Space` / `↑` | Jump |
| Click / Tap | Jump |

## Features

- Pixel art dino & obstacles drawn with vanilla canvas
- Speed increases as your score climbs
- Particle effects on jump & collision
- Hi-score tracking per session
- Mobile-friendly tap support

## Stack

```
HTML  ──  Canvas API  ──  Vanilla JS  ──  CSS
```

Zero dependencies. Single file. Just open and play.

---

## Run Locally

No build step needed — just open the file:

```bash
git clone https://github.com/DihanSae29/dino-jump
cd dino-jump
open dino_goreng.html
```

Or serve it:

```bash
npx serve .
```
